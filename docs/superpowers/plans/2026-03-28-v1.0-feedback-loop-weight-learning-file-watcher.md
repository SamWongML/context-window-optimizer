# v1.0: Feedback Loop, Weight Learning, and File Watcher — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Close the optimization loop — track which packed context was actually useful, learn better scoring weights from feedback, and keep the index fresh via file watching.

**Architecture:** Three subsystems behind existing feature flags (`feedback`, `watch`). The feedback store (SQLite) persists pack sessions and per-file utilization scores. A utilization scorer measures identifier overlap between packed files and LLM responses. An EMA-based learner updates scoring weights from accumulated feedback. A file watcher triggers incremental re-discovery on changes. All new code is conditionally compiled and the default (non-feedback, non-watch) build remains unchanged.

**Tech Stack:** rusqlite 0.32 (bundled SQLite), notify 7 (file watching), existing tiktoken/tree-sitter/rayon stack. No new dependencies needed.

/// A feedback session record.
#[derive(Debug, Clone)]
pub struct Session {
    /// Unique session identifier.
    pub id: String,
    /// Repository path.
    pub repo: String,
    /// Token budget used for this pack.
    pub budget: usize,
    /// Unix timestamp when the session was created.
    pub created_at: i64,
}

/// Per-file feedback record within a session.
#[derive(Debug, Clone)]
pub struct FeedbackRecord {
    /// Path to the file (relative to repo root).
    pub file_path: String,
    /// Token count of the file.
    pub token_count: usize,
    /// Composite score assigned during packing.
    pub composite_score: f32,
    /// Whether this file was selected in the pack result.
    pub was_selected: bool,
    /// Utilization score (0.0–1.0): how much the LLM referenced this file.
    /// `None` if not yet computed.
    pub utilization: Option<f32>,
}

/// Stored weight snapshot from learning.
#[derive(Debug, Clone)]
pub struct WeightSnapshot {
    /// Recency weight.
    pub recency: f32,
    /// Size weight.
    pub size: f32,
    /// Proximity weight.
    pub proximity: f32,
    /// Dependency weight.
    pub dependency: f32,
    /// Average utilization across sessions at time of snapshot.
    pub avg_utilization: f32,
    /// Unix timestamp.
    pub created_at: i64,
}

/// SQLite-backed feedback store.
///
/// Persists pack sessions, per-file utilization feedback, and learned weight history.
///
/// # Examples
///
/// ```no_run
/// use ctx_optim::feedback::store::FeedbackStore;
/// let store = FeedbackStore::open(".ctx-optim/feedback.db").unwrap();
/// ```
pub struct FeedbackStore {
    conn: Connection,
}

impl FeedbackStore {
    /// Open (or create) the feedback database at the given path.
    ///
    /// Creates the parent directory if it doesn't exist.
    pub fn open(path: impl AsRef<Path>) -> Result<Self, OptimError> {
        let path = path.as_ref();
        if let Some(parent) = path.parent() {
            std::fs::create_dir_all(parent).map_err(|e| {
                OptimError::Feedback(format!(
                    "cannot create directory {}: {e}",
                    parent.display()
                ))
            })?;
        }
        let conn = Connection::open(path)
            .map_err(|e| OptimError::Feedback(format!("cannot open database: {e}")))?;
        let store = Self { conn };
        store.create_schema()?;
        Ok(store)
    }

    /// Open an in-memory database (for testing).
    pub fn open_in_memory() -> Result<Self, OptimError> {
        let conn = Connection::open_in_memory()
            .map_err(|e| OptimError::Feedback(format!("cannot open in-memory db: {e}")))?;
        let store = Self { conn };
        store.create_schema()?;
        Ok(store)
    }

    fn create_schema(&self) -> Result<(), OptimError> {
        self.conn
            .execute_batch(
                "
            CREATE TABLE IF NOT EXISTS sessions (
                id          TEXT PRIMARY KEY,
                repo        TEXT NOT NULL,
                budget      INTEGER NOT NULL,
                created_at  INTEGER NOT NULL
            );

            CREATE TABLE IF NOT EXISTS feedback (
                id              INTEGER PRIMARY KEY AUTOINCREMENT,
                session_id      TEXT NOT NULL REFERENCES sessions(id),
                file_path       TEXT NOT NULL,
                token_count     INTEGER NOT NULL,
                composite_score REAL NOT NULL,
                was_selected    INTEGER NOT NULL,
                utilization     REAL,
                UNIQUE(session_id, file_path)
            );

            CREATE TABLE IF NOT EXISTS weight_history (
                id              INTEGER PRIMARY KEY AUTOINCREMENT,
                recency         REAL NOT NULL,
                size            REAL NOT NULL,
                proximity       REAL NOT NULL,
                dependency      REAL NOT NULL,
                avg_utilization REAL NOT NULL,
                created_at      INTEGER NOT NULL
            );
            ",
            )
            .map_err(|e| OptimError::Feedback(format!("schema creation failed: {e}")))?;
        Ok(())
    }
}
```

    /// Insert a new pack session.
    pub fn create_session(&self, session: &Session) -> Result<(), OptimError> {
        self.conn
            .execute(
                "INSERT INTO sessions (id, repo, budget, created_at) VALUES (?1, ?2, ?3, ?4)",
                rusqlite::params![
                    session.id,
                    session.repo,
                    session.budget as i64,
                    session.created_at,
                ],
            )
            .map_err(|e| OptimError::Feedback(format!("insert session: {e}")))?;
        Ok(())
    }

    /// Retrieve a session by ID.
    pub fn get_session(&self, session_id: &str) -> Result<Option<Session>, OptimError> {
        let mut stmt = self
            .conn
            .prepare("SELECT id, repo, budget, created_at FROM sessions WHERE id = ?1")
            .map_err(|e| OptimError::Feedback(format!("prepare: {e}")))?;

        let mut rows = stmt
            .query_map(rusqlite::params![session_id], |row| {
                Ok(Session {
                    id: row.get(0)?,
                    repo: row.get(1)?,
                    budget: row.get::<_, i64>(2)? as usize,
                    created_at: row.get(3)?,
                })
            })
            .map_err(|e| OptimError::Feedback(format!("query session: {e}")))?;

        match rows.next() {
            Some(Ok(session)) => Ok(Some(session)),
            Some(Err(e)) => Err(OptimError::Feedback(format!("read session: {e}"))),
            None => Ok(None),
        }
    }

    /// Record per-file feedback for a session.
    pub fn record_feedback(
        &self,
        session_id: &str,
        records: &[FeedbackRecord],
    ) -> Result<(), OptimError> {
        let tx = self
            .conn
            .unchecked_transaction()
            .map_err(|e| OptimError::Feedback(format!("begin tx: {e}")))?;

        for r in records {
            tx.execute(
                "INSERT OR REPLACE INTO feedback (session_id, file_path, token_count, composite_score, was_selected, utilization)
                 VALUES (?1, ?2, ?3, ?4, ?5, ?6)",
                rusqlite::params![
                    session_id,
                    r.file_path,
                    r.token_count as i64,
                    r.composite_score as f64,
                    r.was_selected as i64,
                    r.utilization.map(|u| u as f64),
                ],
            )
            .map_err(|e| OptimError::Feedback(format!("insert feedback: {e}")))?;
        }

        tx.commit()
            .map_err(|e| OptimError::Feedback(format!("commit: {e}")))?;
        Ok(())
    }

    /// Get all feedback records for a session.
    pub fn get_session_feedback(
        &self,
        session_id: &str,
    ) -> Result<Vec<FeedbackRecord>, OptimError> {
        let mut stmt = self
            .conn
            .prepare(
                "SELECT file_path, token_count, composite_score, was_selected, utilization
                 FROM feedback WHERE session_id = ?1 ORDER BY composite_score DESC",
            )
            .map_err(|e| OptimError::Feedback(format!("prepare: {e}")))?;

        let rows = stmt
            .query_map(rusqlite::params![session_id], |row| {
                Ok(FeedbackRecord {
                    file_path: row.get(0)?,
                    token_count: row.get::<_, i64>(1)? as usize,
                    composite_score: row.get::<_, f64>(2)? as f32,
                    was_selected: row.get::<_, i64>(3)? != 0,
                    utilization: row.get::<_, Option<f64>>(4)?.map(|v| v as f32),
                })
            })
            .map_err(|e| OptimError::Feedback(format!("query feedback: {e}")))?;

        rows.collect::<Result<Vec<_>, _>>()
            .map_err(|e| OptimError::Feedback(format!("read feedback: {e}")))
    }

    /// Save a weight snapshot.
    pub fn save_weights(&self, snap: &WeightSnapshot) -> Result<(), OptimError> {
        self.conn
            .execute(
                "INSERT INTO weight_history (recency, size, proximity, dependency, avg_utilization, created_at)
                 VALUES (?1, ?2, ?3, ?4, ?5, ?6)",
                rusqlite::params![
                    snap.recency as f64,
                    snap.size as f64,
                    snap.proximity as f64,
                    snap.dependency as f64,
                    snap.avg_utilization as f64,
                    snap.created_at,
                ],
            )
            .map_err(|e| OptimError::Feedback(format!("insert weights: {e}")))?;
        Ok(())
    }

    /// Get the most recently saved weight snapshot.
    pub fn latest_weights(&self) -> Result<Option<WeightSnapshot>, OptimError> {
        let mut stmt = self
            .conn
            .prepare(
                "SELECT recency, size, proximity, dependency, avg_utilization, created_at
                 FROM weight_history ORDER BY created_at DESC LIMIT 1",
            )
            .map_err(|e| OptimError::Feedback(format!("prepare: {e}")))?;

        let mut rows = stmt
            .query_map([], |row| {
                Ok(WeightSnapshot {
                    recency: row.get::<_, f64>(0)? as f32,
                    size: row.get::<_, f64>(1)? as f32,
                    proximity: row.get::<_, f64>(2)? as f32,
                    dependency: row.get::<_, f64>(3)? as f32,
                    avg_utilization: row.get::<_, f64>(4)? as f32,
                    created_at: row.get(5)?,
                })
            })
            .map_err(|e| OptimError::Feedback(format!("query weights: {e}")))?;

        match rows.next() {
            Some(Ok(snap)) => Ok(Some(snap)),
            Some(Err(e)) => Err(OptimError::Feedback(format!("read weights: {e}"))),
            None => Ok(None),
        }
    }

    /// Count the total number of sessions.
    pub fn session_count(&self) -> Result<usize, OptimError> {
        let count: i64 = self
            .conn
            .query_row("SELECT COUNT(*) FROM sessions", [], |row| row.get(0))
            .map_err(|e| OptimError::Feedback(format!("count sessions: {e}")))?;
        Ok(count as usize)
    }

    /// Get all feedback records that have utilization scores (for weight learning).
    ///
    /// Returns `(file_path, composite_score, utilization, recency_signal, size_signal, proximity_signal, dependency_signal)` tuples.
    /// Since we don't store per-signal breakdowns in feedback, this returns only
    /// `(composite_score, utilization)` pairs grouped by file.
    pub fn all_feedback_with_utilization(
        &self,
    ) -> Result<Vec<FeedbackRecord>, OptimError> {
        let mut stmt = self
            .conn
            .prepare(
                "SELECT file_path, token_count, composite_score, was_selected, utilization
                 FROM feedback WHERE utilization IS NOT NULL ORDER BY session_id",
            )
            .map_err(|e| OptimError::Feedback(format!("prepare: {e}")))?;

        let rows = stmt
            .query_map([], |row| {
                Ok(FeedbackRecord {
                    file_path: row.get(0)?,
                    token_count: row.get::<_, i64>(1)? as usize,
                    composite_score: row.get::<_, f64>(2)? as f32,
                    was_selected: row.get::<_, i64>(3)? != 0,
                    utilization: row.get::<_, Option<f64>>(4)?.map(|v| v as f32),
                })
            })
            .map_err(|e| OptimError::Feedback(format!("query all feedback: {e}")))?;

        rows.collect::<Result<Vec<_>, _>>()
            .map_err(|e| OptimError::Feedback(format!("read all feedback: {e}")))
    }

use std::collections::HashSet;

/// Common language keywords to filter out of identifier extraction.
const KEYWORDS: &[&str] = &[
    // Rust
    "fn", "let", "mut", "pub", "use", "mod", "struct", "enum", "impl", "trait",
    "where", "for", "loop", "while", "if", "else", "match", "return", "break",
    "continue", "self", "super", "crate", "true", "false", "const", "static",
    "type", "async", "await", "move", "ref", "dyn", "unsafe",
    // TypeScript/JavaScript
    "function", "var", "const", "class", "interface", "export", "import",
    "from", "extends", "implements", "new", "this", "typeof", "instanceof",
    "null", "undefined", "void", "never",
    // Python
    "def", "class", "import", "from", "None", "True", "False", "lambda",
    "with", "as", "try", "except", "finally", "raise", "yield", "pass",
    // Go
    "func", "package", "import", "type", "struct", "interface", "map",
    "chan", "go", "defer", "select", "case", "default", "range", "nil",
];

/// Extract meaningful identifiers from source code.
///
/// Splits on non-alphanumeric/underscore boundaries, filters out:
/// - Tokens shorter than 3 characters
/// - Common language keywords
/// - Pure numeric tokens
///
/// # Examples
///
/// ```
/// use ctx_optim::feedback::utilization::extract_identifiers;
/// let ids = extract_identifiers("fn calculate_score(entry: &FileEntry) -> f32 {}");
/// assert!(ids.contains("calculate_score"));
/// assert!(ids.contains("FileEntry"));
/// ```
pub fn extract_identifiers(source: &str) -> HashSet<String> {
    let keywords: HashSet<&str> = KEYWORDS.iter().copied().collect();

    source
        .split(|c: char| !c.is_alphanumeric() && c != '_')
        .filter(|token| {
            let t = *token;
            t.len() >= 3
                && !t.chars().all(|c| c.is_ascii_digit())
                && !keywords.contains(t)
        })
        .map(|t| t.to_string())
        .collect()
}

/// Compute a utilization score by measuring identifier overlap.
///
/// Returns a value in `[0.0, 1.0]` representing what fraction of the file's
/// identifiers appeared in the LLM response.
///
/// - `1.0` means every identifier in the file was referenced
/// - `0.0` means none were referenced (or file is empty)
///
/// # Examples
///
/// ```
/// use ctx_optim::feedback::utilization::utilization_score;
/// let score = utilization_score(
///     "pub fn compute(data: &Input) -> Output { data.process() }",
///     "I called compute on the Input data to get the Output",
/// );
/// assert!(score > 0.3);
/// ```
pub fn utilization_score(file_content: &str, llm_response: &str) -> f32 {
    let file_ids = extract_identifiers(file_content);
    if file_ids.is_empty() {
        return 0.0;
    }

    let response_tokens: HashSet<String> = llm_response
        .split(|c: char| !c.is_alphanumeric() && c != '_')
        .filter(|t| t.len() >= 3)
        .map(|t| t.to_string())
        .collect();

    if response_tokens.is_empty() {
        return 0.0;
    }

    let overlap = file_ids.intersection(&response_tokens).count();
    (overlap as f32 / file_ids.len() as f32).clamp(0.0, 1.0)
}

```rust
/// Compute updated scoring weights from feedback history.
///
/// The algorithm:
/// 1. Gather all feedback records with utilization scores.
/// 2. Compute the mean utilization for high-scoring vs low-scoring files.
/// 3. For files with high utilization, boost the weight of the dominant signal.
/// 4. Apply EMA: `w_new = alpha * w_target + (1 - alpha) * w_current`.
/// 5. Normalize so weights sum to 1.0.
///
/// Returns the current weights unchanged if no feedback data exists.
///
/// # Examples
///
/// ```no_run
/// use ctx_optim::feedback::store::FeedbackStore;
/// use ctx_optim::config::ScoringWeights;
/// use ctx_optim::feedback::learning::compute_updated_weights;
/// let store = FeedbackStore::open_in_memory().unwrap();
/// let current = ScoringWeights::default();
/// let updated = compute_updated_weights(&store, &current, 0.1).unwrap();
/// ```
pub fn compute_updated_weights(
    store: &FeedbackStore,
    current: &ScoringWeights,
    learning_rate: f32,
) -> Result<ScoringWeights, OptimError> {
    let all_feedback = store.all_feedback_with_utilization()?;

    if all_feedback.is_empty() {
        return Ok(current.clone());
    }

    // Partition feedback into high-util and low-util groups
    let util_threshold = 0.5;
    let (high_util, low_util): (Vec<_>, Vec<_>) = all_feedback
        .iter()
        .filter(|r| r.utilization.is_some())
        .partition(|r| r.utilization.unwrap() >= util_threshold);

    if high_util.is_empty() {
        return Ok(current.clone());
    }

    // Compute signal correlations:
    // High-util files tend to have high composite scores → recency/size signals working well
    // Low-util files with high scores → signals gave false confidence

    let avg_high_score: f32 =
        high_util.iter().map(|r| r.composite_score).sum::<f32>() / high_util.len() as f32;
    let avg_low_score: f32 = if low_util.is_empty() {
        0.0
    } else {
        low_util.iter().map(|r| r.composite_score).sum::<f32>() / low_util.len() as f32
    };

    // Score discrimination: how well does the scoring separate useful from useless?
    // If high-util files already have higher scores, weights are working — nudge gently.
    // If low-util files have higher scores, something is wrong — adjust more.
    let discrimination = (avg_high_score - avg_low_score).clamp(-1.0, 1.0);

    // Compute target weights based on file characteristics in high-util group:
    // - Files with small token counts suggest size signal is valuable
    // - Files with high composite scores suggest current balance is good
    let avg_high_tokens: f32 =
        high_util.iter().map(|r| r.token_count as f32).sum::<f32>() / high_util.len() as f32;
    let avg_low_tokens: f32 = if low_util.is_empty() {
        avg_high_tokens
    } else {
        low_util.iter().map(|r| r.token_count as f32).sum::<f32>() / low_util.len() as f32
    };

    // If high-util files are smaller, boost size weight; if larger, reduce it
    let size_boost = if avg_high_tokens < avg_low_tokens {
        1.2
    } else {
        0.8
    };

    // If discrimination is positive (good), maintain current weights
    // If negative (bad), increase recency weight (most reliable signal)
    let recency_boost = if discrimination < 0.0 { 1.3 } else { 1.0 };

    // Compute target weights
    let target_recency = current.recency * recency_boost;
    let target_size = current.size * size_boost;
    let target_proximity = current.proximity; // unchanged by this heuristic
    let target_dependency = current.dependency; // unchanged by this heuristic

    // EMA blend
    let new_recency = (1.0 - learning_rate) * current.recency + learning_rate * target_recency;
    let new_size = (1.0 - learning_rate) * current.size + learning_rate * target_size;
    let new_proximity =
        (1.0 - learning_rate) * current.proximity + learning_rate * target_proximity;
    let new_dependency =
        (1.0 - learning_rate) * current.dependency + learning_rate * target_dependency;

    // Normalize to sum to 1.0
    let sum = new_recency + new_size + new_proximity + new_dependency;
    let weights = if sum > 0.0 {
        ScoringWeights {
            recency: new_recency / sum,
            size: new_size / sum,
            proximity: new_proximity / sum,
            dependency: new_dependency / sum,
        }
    } else {
        current.clone()
    };

    Ok(weights)
}

/// Run a full learning cycle: compute updated weights and store a snapshot.
///
/// Returns `Some(weights)` if learning produced new weights, `None` if
/// there wasn't enough data.
///
/// # Examples
///
/// ```no_run
/// use ctx_optim::feedback::store::FeedbackStore;
/// use ctx_optim::config::ScoringWeights;
/// use ctx_optim::feedback::learning::run_learning_cycle;
/// let store = FeedbackStore::open_in_memory().unwrap();
/// let current = ScoringWeights::default();
/// if let Some(updated) = run_learning_cycle(&store, &current, 0.1).unwrap() {
///     println!("New weights: recency={:.3}", updated.recency);
/// }
/// ```
pub fn run_learning_cycle(
    store: &FeedbackStore,
    current: &ScoringWeights,
    learning_rate: f32,
) -> Result<Option<ScoringWeights>, OptimError> {
    let all_feedback = store.all_feedback_with_utilization()?;
    if all_feedback.is_empty() {
        return Ok(None);
    }

    let updated = compute_updated_weights(store, current, learning_rate)?;

    // Compute average utilization for the snapshot
    let avg_util: f32 = all_feedback
        .iter()
        .filter_map(|r| r.utilization)
        .sum::<f32>()
        / all_feedback.len().max(1) as f32;

    let snapshot = WeightSnapshot {
        recency: updated.recency,
        size: updated.size,
        proximity: updated.proximity,
        dependency: updated.dependency,
        avg_utilization: avg_util,
        created_at: crate::feedback::unix_now(),
    };
    store.save_weights(&snapshot)?;

    Ok(Some(updated))

/// Score a single `FileEntry` with an optional per-file feedback multiplier.
///
/// If the file's path appears in `feedback_multipliers`, its composite score
/// is multiplied (and clamped to `[0.0, 1.0]`). Files with high utilization
/// in previous sessions get multipliers > 1.0; files with low utilization get < 1.0.
///
/// # Examples
///
/// ```no_run
/// use ctx_optim::{config::ScoringWeights, scoring::score_entry_with_feedback};
/// use std::collections::HashMap;
/// // feedback_multipliers maps file paths to multipliers derived from utilization history
/// let multipliers = HashMap::new();
/// ```
pub fn score_entry_with_feedback(
    entry: &FileEntry,
    weights: &ScoringWeights,
    focus_paths: &[PathBuf],
    dep_graph: Option<&DependencyGraph>,
    feedback_multipliers: &std::collections::HashMap<PathBuf, f32>,
) -> ScoredEntry {
    let mut scored = score_entry(entry, weights, focus_paths, dep_graph);

    if let Some(&multiplier) = feedback_multipliers.get(&entry.path) {
        scored.composite_score = (scored.composite_score * multiplier).clamp(0.0, 1.0);
    }

    scored
}

/// Score a batch of entries in parallel, with optional feedback multipliers.
pub fn score_entries_with_feedback(
    entries: &[FileEntry],
    weights: &ScoringWeights,
    focus_paths: &[PathBuf],
    dep_graph: Option<&DependencyGraph>,
    feedback_multipliers: &std::collections::HashMap<PathBuf, f32>,
) -> Vec<ScoredEntry> {
    if feedback_multipliers.is_empty() {
        return score_entries(entries, weights, focus_paths, dep_graph);
    }
    entries
        .par_iter()
        .map(|e| score_entry_with_feedback(e, weights, focus_paths, dep_graph, feedback_multipliers))
        .collect()
}

#[cfg(feature = "feedback")]
fn load_feedback_context(
    db_path: &Path,
    default_weights: &config::ScoringWeights,
) -> (config::ScoringWeights, std::collections::HashMap<PathBuf, f32>) {
    use feedback::store::FeedbackStore;

    let store = match FeedbackStore::open(db_path) {
        Ok(s) => s,
        Err(e) => {
            tracing::debug!("no feedback store: {e}");
            return (default_weights.clone(), std::collections::HashMap::new());
        }
    };

    // Load learned weights if available
    let weights = match store.latest_weights() {
        Ok(Some(snap)) => config::ScoringWeights {
            recency: snap.recency,
            size: snap.size,
            proximity: snap.proximity,
            dependency: snap.dependency,
        },
        _ => default_weights.clone(),
    };

    // Build per-file feedback multipliers from historical utilization
    let multipliers = match store.all_feedback_with_utilization() {
        Ok(records) => {
            let mut file_utils: std::collections::HashMap<String, Vec<f32>> = std::collections::HashMap::new();
            for r in &records {
                if let Some(u) = r.utilization {
                    file_utils.entry(r.file_path.clone()).or_default().push(u);
                }
            }
            file_utils
                .into_iter()
                .map(|(path, utils)| {
                    let avg: f32 = utils.iter().sum::<f32>() / utils.len() as f32;
                    // Map utilization 0.0–1.0 to multiplier 0.5–1.5
                    let multiplier = 0.5 + avg;
                    (PathBuf::from(path), multiplier)
                })
                .collect()
        }
        Err(e) => {
            tracing::debug!("cannot load feedback multipliers: {e}");
            std::collections::HashMap::new()
        }
    };

    (weights, multipliers)
}

/// Source file extensions that trigger re-indexing.
const WATCHED_EXTENSIONS: &[&str] = &[
    "rs", "ts", "tsx", "js", "jsx", "mjs", "cjs", "py", "pyi", "go",
    "toml", "yaml", "yml", "json",
];

/// Directory names to ignore.
const IGNORED_DIRS: &[&str] = &["target", "node_modules", ".git", "dist", "build", "__pycache__"];

/// Check if a file path has a watched extension.
///
/// # Examples
///
/// ```
/// use ctx_optim::watch::is_watched_extension;
/// assert!(is_watched_extension(std::path::Path::new("src/main.rs")));
/// assert!(!is_watched_extension(std::path::Path::new("image.png")));
/// ```
pub fn is_watched_extension(path: &Path) -> bool {
    path.extension()
        .and_then(|e| e.to_str())
        .is_some_and(|ext| WATCHED_EXTENSIONS.contains(&ext))
}

/// Check if a path falls under an ignored directory.
///
/// # Examples
///
/// ```
/// use ctx_optim::watch::should_ignore_path;
/// assert!(should_ignore_path(std::path::Path::new("target/debug/main")));
/// assert!(!should_ignore_path(std::path::Path::new("src/main.rs")));
/// ```
pub fn should_ignore_path(path: &Path) -> bool {
    path.components().any(|c| {
        c.as_os_str()
            .to_str()
            .is_some_and(|s| IGNORED_DIRS.contains(&s))
    })
}

/// Start watching a directory for source file changes.
///
/// Returns a `RecommendedWatcher` (must be kept alive) and sends batches
/// of changed file paths through the channel.
///
/// Only source files (by extension) are reported. Ignored directories
/// (target/, node_modules/, .git/) are filtered out.
///
/// # Examples
///
/// ```no_run
/// use ctx_optim::watch::start_watching;
/// use std::sync::mpsc;
/// let (tx, rx) = mpsc::channel();
/// let _watcher = start_watching(std::path::Path::new("."), tx).unwrap();
/// // rx will receive Vec<PathBuf> batches of changed source files
/// ```
pub fn start_watching(
    root: &Path,
    sender: mpsc::Sender<Vec<PathBuf>>,
) -> Result<RecommendedWatcher, OptimError> {
    let mut watcher = RecommendedWatcher::new(
        move |result: Result<Event, notify::Error>| {
            if let Ok(event) = result {
                match event.kind {
                    EventKind::Create(_) | EventKind::Modify(_) | EventKind::Remove(_) => {
                        let relevant: Vec<PathBuf> = event
                            .paths
                            .into_iter()
                            .filter(|p| is_watched_extension(p) && !should_ignore_path(p))
                            .collect();
                        if !relevant.is_empty() {
                            let _ = sender.send(relevant);
                        }
                    }
                    _ => {}
                }
            }
        },
        notify::Config::default().with_poll_interval(Duration::from_secs(1)),
    )
    .map_err(|e| OptimError::Watch(format!("create watcher: {e}")))?;

    watcher
        .watch(root, RecursiveMode::Recursive)
        .map_err(|e| OptimError::Watch(format!("watch {}: {e}", root.display())))?;

    tracing::info!("file watcher started on {}", root.display());
    Ok(watcher)
}

/// Run the file watcher loop, logging changed files.
///
/// This blocks the calling thread until the channel is closed.
/// Useful for the CLI `watch` command.
pub fn run_watch_loop(
    root: &Path,
) -> Result<(), OptimError> {
    let (tx, rx) = mpsc::channel();
    let _watcher = start_watching(root, tx)?;

    tracing::info!("watching {} for source file changes (Ctrl+C to stop)", root.display());

    loop {
        match rx.recv() {
            Ok(paths) => {
                for path in &paths {
                    tracing::info!("changed: {}", path.display());
                }
            }
            Err(_) => {
                tracing::info!("watcher channel closed");
                break;
            }
        }
    }

    Ok(())
}
